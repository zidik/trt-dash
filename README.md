# trt-dash 
Dashboard for football robot.
Project is mainly created in order to try out:
* Typescript 2
* Webpack 2
* Hot Module Reload
* React and Redux
### Live example
https://trt-dash.herokuapp.com/
### Getting Started
```bash
# clone the repo
git clone https://github.com/zidik/trt-dash.git

# run install
npm install

# run dev server
npm run dev
```
Open browser at http://localhost:3000
